# AI Landing Page Analyzer

Analyze landing page messaging, CTAs, and trust signals against competitors.

## Live Demo

https://landingtestai.vercel.app

## Setup

```bash
npm install
cp .env.example .env.local
# Add your OPENAI_API_KEY and FIRECRAWL_API_KEY to .env.local
npm run dev
```

## Stack

React, Next.js, Tailwind, OpenAI GPT-5.4-mini, Firecrawl

## Notes

V1 - text-based extraction with JS rendering via Firecrawl v2/scrape. Visual grounding is Next Step #1.
